# M-Tech Internship Tasks & Projects

Welcome to the central repository for my M-Tech Internship work! This repository tracks my progress through various tasks, assignments, and projects assigned during the internship, focusing on Artificial Intelligence, Machine Learning, and foundational Computer Science algorithms.

## 🎯 Purpose and Objectives

The core purpose of this repository is to:
- Document my weekly progress and report submissions.
- Implement foundational programming logic and advanced AI algorithms from scratch.
- Apply theoretical concepts into practical, working Python projects.
- Maintain a structured and clean codebase utilizing software engineering best practices.

### Task 02

### Task 03 (Core Implementation & AI Lab)
This is the most comprehensive section containing both theoretical lab manual completions and functional Python projects.
- **Lab Manual Tasks (`Lab_Manual_Tasks_Completed/`):** Implements algorithms and exercises across 9 core modules:
  - Python Foundations & Data Structures

- **Capstone Project (`capstone_smart_route_planner_Project/`):** A practical application culminating the search algorithms into a smart route-planning script.
- **Project - Smart Study Portal (`Project/smart_study_portal/`):** A mini portal application that helps students explore AI/ML learning modules, track their progress, and view topics, complete with algorithms and unit testing.
- **Report:** `Task_03_Report.docx`, `Task_03_Report.pdf`

## 🌟 Best Practices Followed

Throughout this internship, I am actively adhering to the following best practices to ensure high-quality work:
1. **Clean Code & Modularity:** Writing reusable functions, separating logic from data, and keeping code files organized by domain (e.g., separating `algorithms.py`, `data.py`, and `app.py`).
2. **Version Control:** Making logical commits and pushing code regularly to GitHub to maintain a reliable project history.
3. **Documentation:** Adding inline comments for complex AI algorithms (like A* and UCS) and maintaining clear `README` files for individual sub-projects.
4. **Testing:** Incorporating unit tests (e.g., `test_algorithms.py`) to verify that the core logic and algorithms function correctly.
5. **Algorithmic Efficiency:** Prioritizing optimization in time and space complexity when implementing search algorithms for AI models.
