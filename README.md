## 👩‍💻 About This Repository

This repository documents the complete work done during my **M-Tech AI/ML Internship**, organized week by week. It covers a hands-on progression from the **fundamentals of Artificial Intelligence** (search algorithms, graph traversal, Python programming) all the way to advanced **Machine Learning** concepts including supervised & unsupervised learning, model evaluation, and ensemble methods.

Each week contains:
- 📄 **Task PDFs** — Assignment sheets with objectives and requirements
- 🐍 **Python Scripts** — Clean, well-commented implementations
- 📓 **Jupyter Notebooks** — Interactive experiments and analysis
- 🚀 **Projects** — End-to-end applied AI/ML applications

### 🚑 Day 04 — Mini Project: Emergency Ambulance Routing System

> A real-world AI application that compares multiple search algorithms (BFS, UCS, A*) to find the optimal ambulance route through a city map.

**Features:**
- 🗺️ Dynamic city graph with weighted edges (travel times)
- ⚡ Side-by-side comparison of BFS vs UCS vs A* Search
- 📊 Nodes explored & cost analysis for each algorithm
- 🏆 Automatic recommendation of the best algorithm

```bash
cd "Week 01/Day 04 + Mini Project"
python emergency_routing.py
```

---

### 📊 Day 05 — Weekly Project: Promotional Campaign Effectiveness Analyzer

> An AI-powered desktop application that uses **Random Forest Machine Learning** to analyze marketing campaign data, identify the most effective tactics, and predict outcomes on new customer data.

**Key Features:**
- 📂 Upload any CSV campaign dataset
- 🤖 Random Forest classifier with balanced class weighting
- 📈 Feature importance visualization
- 🔲 Confusion matrix & performance metrics
- 📝 Auto-generated executive business report
- 🔮 Apply model to new customer data for predictions
- 🌙 Premium dark-mode desktop UI (CustomTkinter)

```bash
cd "Week 01/Day 05 + Weekly Project/promotional compaign effectiveness analyzer"
pip install -r requirements.txt
python main.py
```
