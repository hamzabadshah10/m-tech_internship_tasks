# Smart Study Portal

A simple Python mini portal that helps students explore AI/ML learning modules, view topics, and track progress.

## What this project does
- Shows a welcome screen for the student
- Lets the user choose a learning module
- Displays lesson topics for that module
- Tracks completed modules
- Suggests the next module to study

## How to run the Python version
```bash
python main.py
```

## How to run the HTML version
Open the file in a browser:
```text
index.html
```

## Project structure
- portal.py: core portal logic
- main.py: entry point
- index.html: browser-based UI
- styles.css: page styling
- tests/test_portal.py: unit tests
